01: Tổng quan — Giới thiệu bài toán

GIỚI THIỆU ĐỀ TÀI & TÍNH CẤP THIẾT

Bối cảnh

Trong môi trường an ninh mạng hiện đại, mã độc ngày càng tinh vi với các kỹ thuật ngụy trang và biến đổi cấu trúc. Vì vậy, công cụ phân tích cần đọc sâu dữ liệu thô thay vì chỉ dựa vào đuôi tệp như .exe hoặc .doc.

Giải pháp đề xuất

  • Hệ thống sử dụng Python kết hợp với thư viện YARA để phát hiện mã độc bằng kỹ thuật so khớp mẫu.
  • Tích hợp thành một hệ thống tự động có khả năng rà quét các tệp tin đơn lẻ và đệ quy toàn bộ thư mục đối với các vùng dữ liệu đáng ngờ.
02 — Công cụ phân tích

TỔNG QUAN VỀ CÔNG CỤ YARA

Khái niệm

YARA là gì?

  • YARA là công cụ mã nguồn mở, dùng để tạo quy tắc mô tả mã độc dựa trên chuỗi văn bản hoặc mẫu nhị phân.
  • Hiện do VirusTotal bảo trợ và phát triển liên tục.
Cơ chế hoạt động

Khớp dấu hiệu trên dữ liệu thô

YARA phân loại và nhận diện mã độc dựa trên các dấu hiệu nhận dạng (Signature-based matching). Công cụ quét, so sánh dữ liệu thô của tệp tin với các mô tả logic do người dùng định nghĩa: chuỗi văn bản, mẫu nhị phân Hex hoặc biểu thức chính quy.

04 — Kiến trúc tổng thể

KIẾN TRÚC HỆ THỐNG TỔNG THỂ

Tầng Giao diện

User Interface

Giao diện dòng lệnh (CLI), hỗ trợ truyền tham số để nạp tệp tin đơn lẻ hoặc quét đệ quy thư mục..

Tầng Ứng dụng

Application Layer

  • Sử dụng Python để biên dịch (compile) các quy tắc YARA.
  • Điều phối luồng xử lý: Lớp 1 (Mã băm) → Lớp 2 (YARA).
Tầng Dữ liệu & Đầu ra

Output Layer

Lưu trữ dữ liệu tại hệ quản trị cơ sở dữ liệu PostgreSQL, trích xuất báo cáo rà quét dưới định dạng Text và hiển thị cảnh báo trực tiếp trên màn hình.

03 — Cấu trúc

CẤU TRÚC MỘT TẬP LUẬT YARA

Một quy tắc (Rule) YARA tiêu chuẩn được chia thành 3 thành phần cốt lõi:

Meta (Siêu dữ liệu)

Lưu trữ thông tin về tác giả, mô tả và ngày tạo của quy tắc.

Strings (Chuỗi dấu hiệu)

Định nghĩa các chuỗi văn bản (Text), chuỗi thập lục phân (Hex) hoặc biểu thức chính quy (Regex).

Condition (Điều kiện logic)

Các phép toán logic quyết định khi nào quy tắc này được kích hoạt.

Ví dụ minh họa cấu trúc cơ bản:

rule MalwareExample { meta: author = "Hồ Quốc Long — Nguyễn Bá Thiều Khôi Nguyên" description = "Phát hiện đoạn mã độc cụ thể" strings: $s1 = "malicious_code" condition: $s1 }
05 — Tích hợp công cụ

Tích hợp YARA vào Môi trường Python

Gói thư viện: Sử dụng yara-python, được cài đặt qua trình quản lý gói pip của Python.

Biên dịch (Compile)

Chuyển đổi tệp luật văn bản (.yar) thành đối tượng thực thi trong bộ nhớ (RAM) nhằm tối ưu hóa tốc độ xử lý.

So khớp (Match)

Áp dụng đối tượng biên dịch lên tệp tin hoặc luồng dữ liệu thô để phát hiện vi phạm.

Mã nguồn minh họa:

import yara # Bước 1: Biên dịch tập luật vào bộ nhớ rules = yara.compile('rules.yar') # Bước 2: Rà quét tệp tin khả nghi matches = rules.match('suspicious.exe') print(matches) # Trả về danh sách quy tắc vi phạm

05 — YARA Rules

Ví dụ: Rule phát hiện Emotet

rule BankingTrojan_Emotet { meta: author = "Nguyễn Bá Thiều Khôi Nguyên - Hồ Quốc Long" malware_family = "Emotet" severity = "high" strings: $id_emotet = "Emotet" ascii wide nocase $ps_cmd = "powershell" ascii wide nocase $ps_window = "-w hidden" ascii wide nocase $ps_enc = "-enc" ascii wide nocase $ps_web = "Net.WebClient" ascii wide nocase $ps_down = "DownloadFile" ascii wide nocase $net_wp = "/wp-content/" ascii wide nocase condition: pe.is_pe and ( $id_emotet or (2 of ($ps_*) and 1 of ($net_*)) ) }
06 — Archive

Cơ chế quét Archive

Đặc điểm

  • Hỗ trợ ZIP và 7z
  • Extract vào memory — không ghi ra disk
  • Quét từng file bằng YARA rules
  • Hỗ trợ nested archives (depth ≤ 3)

Bảo vệ

Archive BombRatio ≤ 100:1
Nested DepthTối đa 3 lớp
File CountTối đa 1000
Size LimitTối đa 100MB
07 — Database

PostgreSQL Database

Cơ sở dữ liệu: yara_malware_signatures

Bảng malware_signatures

  • Lưu malware signatures từ API
  • SHA256 hash (UNIQUE)
  • MD5, SHA1, SHA3-384 hashes
  • Index cho truy vấn nhanh

Bảng scan_results

  • Lịch sử quét
  • Detection method: HASH_MATCH, YARA_MATCH, CLEAN
  • Timestamp tự động

Nguồn dữ liệu: MalwareBazaar API (abuse.ch) — lấy signatures cho 14 họ malware
