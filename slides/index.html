01 — Giới thiệu

Tổng quan dự án

YARA Malware Scanner là hệ thống phát hiện mã độc kết hợp hai phương pháp mạnh mẽ:

Hash-based Detection

So sánh SHA256 hash của file với database malware đã biết. Tốc độ O(1), độ tin cậy cao — khớp chính xác 100%.

YARA Pattern Matching

Quét file bằng các rule YARA để phát hiện malware patterns, variants và families chưa có trong database.

Tính năng chính: Quét file/thư mục • Hỗ trợ ZIP, 7z • 14 họ malware • Báo cáo TXT, JSON, CSV

04 — Phát hiện

Ba lớp phát hiện mã độc

Lớp 1: Hash-based Detection

Tính SHA256 hash → Kiểm tra PostgreSQL database → Tốc độ O(1) — Độ tin cậy cao (khớp chính xác)

Lớp 2: YARA Pattern Matching

Compile YARA rules vào RAM (một lần) → Quét bằng pattern matching → Phát hiện malware variants và families

Lớp 3: Archive Scanning

Extract contents vào memory → Quét từng file bên trong → Hỗ trợ nested archives (depth ≤ 3)

06 — Archive

Cơ chế quét Archive

Đặc điểm

  • Hỗ trợ ZIP và 7z
  • Extract vào memory — không ghi ra disk
  • Quét từng file bằng YARA rules
  • Hỗ trợ nested archives (depth ≤ 3)

Bảo vệ

Archive BombRatio ≤ 100:1
Nested DepthTối đa 3 lớp
File CountTối đa 1000
Size LimitTối đa 100MB
07 — Database

PostgreSQL Database

Cơ sở dữ liệu: yara_malware_signatures

Bảng malware_signatures

  • Lưu malware signatures từ API
  • SHA256 hash (UNIQUE)
  • MD5, SHA1, SHA3-384 hashes
  • Index cho truy vấn nhanh

Bảng scan_results

  • Lịch sử quét
  • Detection method: HASH_MATCH, YARA_MATCH, CLEAN
  • Timestamp tự động

Nguồn dữ liệu: MalwareBazaar API (abuse.ch) — lấy signatures cho 14 họ malware
